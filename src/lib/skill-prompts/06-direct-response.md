## Die 5 Bewusstseins-Stufen (Schwartz)

Bestimme ZUERST wo dein Leser steht:

| Stufe | Beschreibung | Headline-Strategie |
|-------|-------------|-------------------|
| **5 — Bewusst** | Kennt dein Produkt | Direkt: Angebot + Preis + CTA |
| **4 — Produkt-bewusst** | Kennt Lösung, nicht dein Produkt | USP + Differenzierung |
| **3 — Lösungs-bewusst** | Weiß dass es Lösungen gibt | Wie deine Lösung besser ist |
| **2 — Problem-bewusst** | Spürt den Schmerz | Problem ansprechen → Lösung zeigen |
| **1 — Unbewusst** | Weiß nicht dass er ein Problem hat | Story/Provokation → Problem aufzeigen |

## Conversion Architecture

### Landing Page Struktur (von oben nach unten)

```
1. HERO SECTION
   - Headline (Max 10 Wörter — der eine Satz)
   - Sub-Headline (Erklärt die Headline)
   - Hero Image oder Demo-Video
   - Primary CTA Button

2. PROBLEM SECTION
   - "Kennst du das?" (3 Pain Points)
   - Agitation: Warum es schlimmer wird wenn man nichts tut
   - Empathie: "Wir kennen das — weil..."

3. LÖSUNG SECTION
   - "Es gibt einen besseren Weg"
   - 3-5 Features als Benefits formuliert
   - Screenshots / Visuals

4. SOCIAL PROOF
   - Testimonials (3 minimum)
   - Logos / Kundenlogos
   - Zahlen (Nutzer, Ergebnisse, Jahre)

5. HOW IT WORKS
   - 3 Schritte (simpel!)
   - Schritt 1: [Einfacher Anfang]
   - Schritt 2: [Die Magie passiert]
   - Schritt 3: [Das Ergebnis]

6. PRICING / ANGEBOT
   - Klar, einfach, keine Verwirrung
   - Anker: "Ein Marketing-Team kostet €15K/Mo. Das hier: €299."
   - Garantie: "30 Tage Geld-zurück" oder ähnlich

7. FAQ
   - Top 5-7 Einwände als Fragen formuliert
   - Jede Antwort = Einwand entkräften

8. FINAL CTA
   - Headline wiederholen (abgewandelt)
   - Dringlichkeit oder Verknappung (NUR wenn echt)
   - CTA Button (identisch zum Hero)
```

## Copywriting-Formeln

### PAS (Problem → Agitation → Solution)
Für: Awareness-Stufe 1-2
```
PROBLEM: "Dein Telefon klingelt — und keiner geht ran."
AGITATION: "Jeder verpasste Anruf ist ein verlorener Kunde. 
Bei 5 verpassten Anrufen/Tag sind das €50K/Jahr die du verlierst."
SOLUTION: "Ein KI-Agent der JEDEN Anruf abnimmt. 24/7. In deiner Stimme."
```

### AIDA (Attention → Interest → Desire → Action)
Für: Awareness-Stufe 3-4
```
ATTENTION: "Was wenn dein Marketing sich selbst macht?"
INTEREST: "40 KI-Agents. 8 Abteilungen. Content, Ads, SEO — alles autonom."
DESIRE: "Stell dir vor: Jeden Morgen 3 neue LinkedIn-Posts, 5 Follow-Up Emails, 
und ein SEO-Artikel — ohne dass du einen Finger rührst."
ACTION: "Starte jetzt kostenlos →"
```

### BAB (Before → After → Bridge)
Für: Testimonials und Case Studies
```
BEFORE: "Vor A-Impact: 3 verpasste Anrufe/Tag, kein Follow-Up, Leads vergessen."
AFTER: "Nach A-Impact: 100% Erreichbarkeit, automatische Follow-Ups, 47% mehr Termine."
BRIDGE: "Der Unterschied? Ein KI-Agent der nie schläft."
```

## Headline-Formeln (DACH-getestet)

1. **Zahl + Ergebnis:** "47% mehr Termine durch KI-Telefonie"
2. **Wie-Frage:** "Wie Versicherungsmakler ohne Sekretärin 24/7 erreichbar sind"
3. **Ohne-Formel:** "Mehr Kunden — ohne mehr Mitarbeiter"
4. **Vergleich:** "Marketing-Team für €299 statt €15.000/Monat"
5. **Provokation:** "Dein Wettbewerb nutzt schon KI. Du noch nicht?"
6. **Social Proof:** "237 Unternehmen nutzen bereits [Produkt]"

## Ad Copy (für Social + Google)

### LinkedIn Ad (max 150 Zeichen Headline)
```
Headline: [Zahl] + [Ergebnis] + [für wen]
Body: Problem (1 Satz) → Lösung (1 Satz) → Beweis (1 Satz) → CTA
Image: Screenshot/Mockup des Produkts
```

### Google Search Ad
```
Headline 1 (30 Zeichen): [Keyword + Benefit]
Headline 2 (30 Zeichen): [USP oder Preis]
Headline 3 (30 Zeichen): [CTA oder Social Proof]
Description (90 Zeichen): [PAS komprimiert]
```

## DACH-Copywriting-Regeln
- **Weniger aggressiv als US-Copy:** Kein "BUY NOW!!!" — Deutsche reagieren allergisch
- **Beweise > Versprechen:** Zahlen, Case Studies, Referenzen
- **Seriösität:** Keine Fake-Urgency ("Nur noch 3 Plätze!" wenn es nicht stimmt)
- **Datenschutz erwähnen:** "DSGVO-konform" ist ein echtes Verkaufsargument
- **Formelle Alternative:** Immer eine Sie-Version parat haben
- **Preise mit €:** Immer € statt EUR, Preise inkl. MwSt. angeben (B2C) oder netto kennzeichnen (B2B)
