## 9 Newsletter-Formate

### 1. Der Kurator
Beste Links/Ressourcen der Woche, kommentiert.
```
📬 [Name] Weekly — KW [X]
- 🔗 [Link 1] — Warum das wichtig ist (2 Sätze)
- 🔗 [Link 2] — Mein Take dazu (2 Sätze)
- 🔗 [Link 3] — Was das für dich bedeutet (2 Sätze)
💡 Mein Tipp der Woche: [Actionable Tip]
```

### 2. Der Storyteller
Eine Story pro Ausgabe, mit Lektion.
```
📖 [Spannende Headline die neugierig macht]
[Geschichte: 300-500 Wörter]
[Die Lektion: 2-3 Sätze]
[CTA: Was du jetzt tun kannst]
```

### 3. Der Experte
Tiefes Wissen zu einem Thema.
```
🧠 Deep Dive: [Thema]
[Problem erklären: 100 Wörter]
[Lösung/Framework: 200-400 Wörter]
[Beispiel: 100 Wörter]
[Sofort umsetzbar: 3 Action Items]
```

### 4. Das Update
Was ist passiert, was kommt.
```
📊 [Firma] Update — [Monat]
✅ Was wir gemacht haben: [3 Bulletpoints]
📈 Zahlen: [Relevante Metriken]
🔜 Was kommt: [Nächste Schritte]
💬 Frage an dich: [Engagement-Frage]
```

### 5. Der Vergleicher
A vs B — mit klarer Empfehlung.
```
⚔️ [A] vs [B] — Was ist besser für [Zielgruppe]?
[Kriterium 1]: A gewinnt weil...
[Kriterium 2]: B gewinnt weil...
[Kriterium 3]: Unentschieden weil...
🏆 Unser Urteil: [Klare Empfehlung]
```

### 6. Die Checkliste
Actionable, sofort nutzbar.
```
✅ [X]-Punkte Checkliste: [Thema]
□ Punkt 1 — [Warum + Wie]
□ Punkt 2 — [Warum + Wie]
...
📥 Download als PDF: [Link]
```

### 7. Das Interview
Q&A mit interessanter Person.
```
🎤 Im Gespräch mit [Name], [Titel]
F: [Frage 1]
A: [Antwort — gekürzt, auf den Punkt]
...
💡 Key Takeaway: [1 Satz]
```

### 8. Der Zahlenmensch
Daten, Statistiken, Insights.
```
📊 [X] Zahlen die du diese Woche kennen musst
1️⃣ [Statistik] — Was das bedeutet
2️⃣ [Statistik] — Warum das wichtig ist
3️⃣ [Statistik] — Was du daraus machst
```

### 9. Der Persönliche
Ehrlich, verletzlich, authentisch.
```
💭 [Persönliche Headline]
[Ehrliche Geschichte/Erfahrung: 300-500 Wörter]
[Was ich gelernt habe]
[Wie du davon profitieren kannst]
```

## Subject Line Formeln

Für JEDE Newsletter-Ausgabe, liefere 3 Subject-Line Varianten:

1. **Neugier:** "Warum ich [unerwartetes] gemacht habe"
2. **Nutzen:** "[Ergebnis] in [Zeitraum] — so geht's"
3. **Dringlichkeit:** "[Trend] verändert [Branche] — bist du ready?"

**DACH-Regeln für Subject Lines:**
- Keine Emojis im Subject (senkt Open Rate in DE)
- Max 50 Zeichen (Mobile!)
- Kein Clickbait — Deutsche bestrafen das mit Abmeldung
- Kleinschreibung wirkt persönlicher

---

## Timing (DACH)
- **B2B:** Dienstag oder Donnerstag, 09:00-10:00 Uhr
- **B2C:** Samstag 10:00 oder Sonntag 18:00
- **Frequenz:** Wöchentlich (ideal) oder alle 2 Wochen (minimum)
